素数

质数是指在大于1的自然数中，除了1和它本身以外不再有其他因数的自然数。

(1)常规方法（复杂度 O（n根号n））
bool isprime(int n){
if(n<=1) return false;
for(int i=2;i<=sqrt(n);i++){
if(n%i) return false;
}
return true;
}
(2)埃氏筛法（复杂度 O（nloglogn））
从小到大枚举所有数，对每一个素数，筛去他的所有倍数，剩下的就是素数。
某数a如果没有被前面的数筛掉，就是素数。
int prime[maxn],num=0;
bool p[maxn]={false};
void iprime(){
for(int i=2;i<maxn;i++){
if(p[i]==false){
prime[num++]=i;
for(int j=2*i;j<maxn;j=j+i){
p[j]=true;
}
}
}
}
